  • RMBMS = 管理员(manager) + 仓库(database)
  • database = N个table
  • table：
    1. 表结构：定义表的列名和类型

SQL

是什么？

结构化查询语言（Structure Query Language），客户端用来操作服务器

标准

由ISO制定标准，对DMS的统一操作方式，如SQL99

分类

  1. DDL(Data Definition Language)
    增删改 库、表结构！
  2. DML(Data Manipulation Language)
    增删改 表记录！
  3. DCL(Data Control Language)
    访问权限、安全级别、用户创建
  4. DQL(Data Query Language)
    查询记录

DDL

  1. 数据库
  • SHOW DATABASES 查看所有数据库
  • USE　<数据库名>
  • CREATE DATABASE [IF NOT EXISTS] mydb1 [charset=utf-8] [可选，使用不写中括号] 创建数据库
  • DROP DATABASE [if exits] mydb1 删除数据库
  • ALTER DATABASE mydb1 CHARACTER SET utf-8
  1. 数据类型
  • int 整数
  • double double(5, 2) 最多5位，必有2位小数 浮点数
  • decimal 浮点数/十进制、钱相关
  • char 固长字符串(自动补足) 最大255 char(255)
  • varchar 可变长度字符串类型 varchar(65535) ps.需要额外空间记录长度
  • text(标准使用：clob) 字符串类型 最大 2^32 = 4GB
  • blob 字节类型
  • date 日期类型 格式yyyy-MM-dd
  • time 时间类型 格式hh:mm:aa
  • timestamp 时间戳类型
