python基本概念速记

发表于 2018-03-19 | 分类于 笔记

字符编码

python3 的字符串用Unicode编码，一个字符2字节(16位，用4个十六进制)

为节省存储或者便于网络存储，需要把str转换成以字节为单位的bytes

python对bytes类型的数据用带b前缀来表示

str、bytes转换

str -> bytes

bytes -> str

len()长度

len(str) -> str有多少字符

len(bytes) -> bytes 有多少字节

容器

容器，装数据的器具，python内置的数据结构。主要为序列和词典。

序列

  • 列表。['s','d', 'f'] or [3, 23, 5, 9]

  • 元组。('a', 'b', 'c') or (1, 2, 3)。与列表的区别是，元组内的元素不能更改。

  • 字符串。'abcde'

之所以称为序列，是因为其内的元素都有自己的序号–即是元素的位置，“索引”。

序列的通用操作

  1. 索引
  2. 分片
  3. 序列相加
  4. 成员资格

列表操作

列表的独有操作。

  1. list函数(获取)

  2. 元素赋值，删除 位置 a[i] = xxx

  1. 序列的方法 序列.方法(参数)

  • 追加元素 a.append('s')列表最后加一个

  • 弹出位置(元素) a.pop()无参数，弹出最后一个元素； a.pop(i)弹出索引为i的元素

  • 查找元素(第一个) a.index('a') 返回索引，或者异常 (ps. 字符串有个类似的方法find , 找不到返回 -1 )

  • 删除元素(第一个) a.remove('a') 返回索引，或者异常
